  <name>Task 3: Deploy observability to GKE</name>
  <files>infra/k8s/helm-values/prometheus-grafana-gke.yaml, infra/k8s/system/jaeger/deployment-gke.yaml, infra/k8s/system/langfuse/deployment-gke.yaml</files>

  - Langfuse: connect to Cloud SQL instead of local PG, ClusterIP service
  - Create GKE-specific Helm values: infra/k8s/helm-values/prometheus-grafana-gke.yaml

  Apply via kubectl/helm to GKE context.
  Add invoke task: deploy_gke (deploys everything to GKE)
  </action>

</task>

<task type="auto">
  <name>Task 4: Traefik ingress + OAuth2 Proxy for GKE auth</name>
  <files>infra/k8s/helm-values/traefik.yaml, infra/k8s/system/oauth2-proxy/deployment.yaml, infra/k8s/system/oauth2-proxy/service.yaml</files>
  <action>
  Deploy Traefik as K8s ingress controller on GKE:
  - Helm chart: traefik/traefik with infra/k8s/helm-values/traefik.yaml
  - Configure IngressRoute CRDs for all services (Chainlit, API, Grafana, Langfuse, Jaeger, Argo)
  - Cloud Load Balancer created automatically by Traefik's LoadBalancer service

  Deploy OAuth2 Proxy for Google OAuth authentication:
  - infra/k8s/system/oauth2-proxy/deployment.yaml: oauth2-proxy/oauth2-proxy image
  - Configure with Google OAuth provider:
    - OAUTH2_PROXY_PROVIDER=google
    - OAUTH2_PROXY_CLIENT_ID / CLIENT_SECRET from Secret Manager
    - OAUTH2_PROXY_EMAIL_DOMAINS=* (or restrict to specific domains)
    - OAUTH2_PROXY_COOKIE_SECRET (random 32-byte secret)
  - Create Google OAuth credentials in GCP Console (APIs & Services → Credentials → OAuth 2.0 Client ID)
  - Set redirect URI to https://{domain}/oauth2/callback

  Wire Traefik ForwardAuth middleware:
  - Create Traefik Middleware CRD: forwardAuth pointing to oauth2-proxy service
  - Apply middleware to all IngressRoutes

  Local Kind does NOT get Traefik or OAuth2 Proxy — localhost access only, no auth needed.
  </action>
  <verify>kubectl --context gke_helpful-rope-230010_us-central1_null-realm get pods -n null-realm | grep -E "(traefik|oauth2)" && curl -I https://{ingress-ip}/</verify>
  <done>Traefik routes all services, OAuth2 Proxy requires Google login, unauthenticated requests redirected to Google sign-in</done>
</task>
