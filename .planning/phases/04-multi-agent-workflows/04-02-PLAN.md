  <action>
  Build workflow execution engine:
  - nullrealm/orchestrator/workflow_executor.py:
    - WorkflowExecutor class
    - execute_workflow(workflow_name, user_input, session_id):
      1. Fetch workflow from registry
      2. For each step: build Argo WorkflowTemplate with assistant config
      3. Handle depends_on for sequential execution
      4. Handle parallel_with for parallel steps
      5. Pass artifacts between steps (step output → next step input)

  - nullrealm/api/routes/workflows.py:
    - POST /api/v1/workflows/{name}/execute: trigger workflow from API
    - GET /api/v1/workflows/{name}/status/{run_id}: check status

  Wire into Chainlit:
  - Detect workflow-triggering messages (e.g., "Add OAuth to service-a")
  - Trigger workflow execution via API
